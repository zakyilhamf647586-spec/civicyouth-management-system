# AI Content Studio Roadmap

## Current Status

AI Content Studio is already functional as a portfolio feature.

The current version supports:

- Uploading multiple images
- Saving content drafts
- Generating AI-based social media text
- Demo fallback mode when API quota is unavailable
- Multi-provider AI configuration
- Permanent Instagram feed template
- Exporting generated feed image as PNG

## Permanent Template Direction

The final visual template direction is focused on Instagram portrait feed:

- Size: 1080 x 1350 px
- Ratio: 4:5
- Fixed logo placement
- Fixed organization name
- Fixed Instagram footer
- User-based input for title, date, time, location, and activity description
- Image placement based on upload order

## Fixed Brand Elements

These elements should not be changed dynamically:

- Logo: Karang Taruna RW 01
- Organization name: KARANG TARUNA RW 01 RANDUGARUT
- Footer: @kartar.rw01.randugarut
- Main colors: navy, gold, cream
- Feed ratio: 1080 x 1350 px

## Future Improvements

### Visual Refinement

- Improve top header ornament
- Improve title spacing and typography
- Make only selected words such as "RW 01" gold
- Add Instagram icon in footer
- Improve bottom decorative lines
- Improve photo frame consistency
- Improve text spacing in the activity description area

### Image Placement

Current version:

- First uploaded image becomes the main image
- Second uploaded image becomes small image 1
- Third uploaded image becomes small image 2

Future version:

- Allow users to choose image placement manually
- Add drag-and-drop image ordering
- Add preview before rendering final feed

Future version may include:

- Template preview

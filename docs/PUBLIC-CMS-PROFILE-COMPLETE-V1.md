# Public CMS Profile Complete V1

## Scope

Fase 2C memperluas halaman Profil agar seluruh isi utama dapat
kelola melalui CMS terstruktur tanpa mengubah route atau desain
utama website publik.

## Section yang dikelola

- Hero Profil;
- kartu brand GARDA 01;
- Cerita Organisasi;
- Identitas Resmi;
- Visi dan Misi;
- Nilai Organisasi;
- CTA Jelajahi Program;
- SEO title dan meta description.

## Prinsip data

Migration hanya menambahkan section dan field yang belum tersedia.
Konten draft atau published yang sudah pernah diedit tidak ditimpa.

## Alur

```text
Simpan Draft
→ Preview Draft
→ Publikasikan
```

## URL

```text
/website/pages/edit/profile
/website/pages/preview/profile
/profil
```

## Toggle section

Cerita Organisasi, Identitas Resmi, Visi dan Misi, Nilai Organisasi,
dan CTA dapat disembunyikan pada draft maupun versi publik.

## Catatan

Daftar misi ditulis satu butir per baris. Logo tetap berasal dari
Pengaturan Website agar identitas visual tidak diduplikasi.

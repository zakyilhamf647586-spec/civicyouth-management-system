# Panduan Ringkas Publikasi Sosial GARDA 01

## Perbedaan utama

### Data Kegiatan

Digunakan untuk data organisasi dan konten yang dapat tampil pada
website publik melalui halaman kegiatan.

### Publikasi Sosial

Digunakan untuk menyiapkan konten Instagram:

1. memilih kegiatan;
2. menyusun hook dan caption;
3. memilih master Canva;
4. mengelola PIC dan reviewer;
5. menentukan target tayang;
6. mencatat tautan Instagram setelah posting manual.

Publikasi Sosial tidak otomatis membuat berita website dan tidak
otomatis mengunggah konten ke Instagram.

## Alur paling sederhana

```text
Data Kegiatan
→ Buat Konten dari Kegiatan
→ Salin Master Canva
→ Review
→ Posting Manual ke Instagram
→ Catat URL Instagram
```

## Fitur lanjutan

Deadline, analitik, rekomendasi waktu tayang, audit trail, dan
AI Content Studio tetap tersedia, tetapi ditempatkan dalam panel
Fitur Lanjutan agar tidak mengganggu alur utama.

## Perubahan antarmuka

- halaman utama berfokus pada empat aksi;
- status diberi deskripsi tindakan;
- filter dibuat dapat dibuka/tutup;
- tabel diringkas menjadi enam kolom;
- form menampilkan field utama;
- field strategi lanjutan menjadi opsional;
- tersedia halaman `/publications/guide`.

## Dampak

Tidak mengubah database, workflow, RBAC, website publik, atau cara
posting Instagram. Perubahan terutama pada tampilan dan penjelasan.
